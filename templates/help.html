소개

FocusMate란?

FocusMate는 AI 기반 학습 조교 시스템으로, 학습자의 감정 상태를 실시간으로 분석하고 맞춤형 학습 조언을 제공합니다. 학습 효율을 높이고 집중력을 향상시키기 위해 설계되었습니다.

사용 방법

웹캠을 통한 실시간 감정 분석 또는 이미지 업로드를 통해 현재 학습 상태를 확인하고, AI 조교와 대화하여 맞춤형 학습 조언을 받을 수 있습니다. 학습 기록을 통해 감정 상태 변화를 추적할 수도 있습니다.

주요 기능

실시간 감정 분석

웹캠을 통해 학습자의 표정을 분석하여 집중, 흥미, 불안, 피로, 중립 등의 감정 상태를 감지합니다.

맞춤형 학습 조언

감지된 감정 상태에 맞는 학습 전략과 팁을 제공합니다. 집중 상태에는 복잡한 문제 해결을, 피로 상태에는 휴식을 권장하는 등 상황에 맞는 조언을 받을 수 있습니다.

AI 조교와 대화

궁금한 학습 내용이나 학습 방법에 대해 AI 조교와 대화할 수 있습니다. AI는 학습자의 현재 감정 상태를 고려한 맞춤형 답변을 제공합니다.

학습 기록 관리

학습 시간 동안의 감정 상태 변화와 AI 조교와의 대화 내용을 기록하고 확인할 수 있습니다.

감정 감지

감지 시작 방법

메인 화면에서 '감정 감지 시작' 버튼을 클릭하거나 이미지를 업로드하여 감정 분석을 시작할 수 있습니다.

  1. 웹캠 접근 권한을 허용합니다.
  2. '감정 감지 시작' 버튼을 클릭합니다.
  3. 분석 결과가 화면에 표시됩니다.

감지 중지 방법

감정 감지 중 '감지 중단' 버튼을 클릭하여 웹캠 사용을 중지할 수 있습니다.

감정 상태 종류

  • 집중: 학습에 가장 이상적인 상태, 새로운 정보 처리에 최적
  • 흥미: 호기심과 학습 동기가 높은 상태, 새로운 지식 습득에 효과적
  • 불안: 시험이나 과제에 대한 압박감을 느끼는 상태
  • 피로: 학습 효율이 저하될 수 있는 상태, 휴식이 필요
  • 중립: 안정적이고 균형 잡힌 감정 상태

AI 조교와 대화

대화 시작하기

오른쪽 채팅 패널에서 AI 조교와 대화할 수 있습니다:

  1. 메시지 입력란에 질문이나 문제를 입력합니다.
  2. Enter 키를 누르거나 '전송' 버튼을 클릭합니다.
  3. AI 조교의 답변이 채팅창에 표시됩니다.

효과적인 질문 방법

AI 조교에게 효과적으로 질문하기 위한 팁:

  • 구체적이고 명확한 질문을 작성합니다.
  • 복잡한 질문은 여러 단계로 나누어 물어봅니다.
  • 학습 과목이나 주제를 명시하면 더 정확한 답변을 받을 수 있습니다.

대화 기록

AI 조교와의 대화 내용은 세션 동안 저장되며, 페이지를 새로고침하거나 브라우저를 닫으면 초기화됩니다.

학습 기록

기록 확인 방법

오른쪽 탭에서 '감정 기록' 탭을 클릭하여 감지된 감정 상태 기록을 확인할 수 있습니다. 현재 버전에서는 세션 동안의 기록만 제공됩니다.

필터 사용 방법

히스토리 페이지 상단의 필터를 사용하여 특정 기간이나 감정 상태의 데이터만 확인할 수 있습니다:

  • 기간 필터: 오늘/이번 주/이번 달/전체 중 선택할 수 있습니다.
  • 감정 필터: 특정 감정 상태의 기록만 볼 수 있습니다.

데이터 내보내기

학습 데이터를 JSON 형식으로 내보내서 백업하거나 외부 분석에 활용할 수 있습니다:

  1. 히스토리 페이지 상단의 "내보내기" 버튼을 클릭합니다.
  2. 데이터 파일이 자동으로 다운로드됩니다.

설정

일반 설정

  • 테마: 라이트 모드, 다크 모드, 또는 시스템 설정에 따름 중 선택할 수 있습니다.
  • 언어: 인터페이스 언어를 변경할 수 있습니다.

분석 설정

  • 감정 감지 민감도: 높음/중간/낮음 중에서 선택하여 감정 분석의 민감도를 조정할 수 있습니다.
  • 감정 감지 간격: 자동 감지 시 촬영 간격을 2초/3초/5초/10초 중에서 선택할 수 있습니다.

학습 설정

  • AI 조교 스타일: 조교의 응답 스타일을 '직설적', '친절한', '상세한' 중에서 선택할 수 있습니다.
  • 자동 추천: 감정 상태에 따른 자동 학습 추천 기능을 켜거나 끌 수 있습니다.

데이터 관리

  • 데이터 삭제: 저장된 모든 감정 기록과 대화 내용을 삭제할 수 있습니다.
  • 자동 저장: 대화 및 감정 상태 자동 저장 기능을 켜거나 끌 수 있습니다.

개인정보 및 보안

개인정보 보호

FocusMate는 사용자의 개인정보를 최우선으로 보호합니다:

  • 모든 얼굴 분석은 사용자의 장치에서 로컬로 처리됩니다.
  • 카메라 이미지는 서버로 전송되지 않으며, 분석 후 즉시 폐기됩니다.
  • 대화 내용은 암호화되어 저장되며, 사용자의 요청 시 완전히 삭제할 수 있습니다.

문제 해결

카메라 접근 문제

카메라에 접근할 수 없는 경우:

  1. 브라우저 설정에서 카메라 권한을 확인하세요.
  2. 다른 프로그램이 카메라를 사용 중인지 확인하세요.
  3. 페이지를 새로고침하거나 브라우저를 재시작해보세요.

AI 응답 문제

AI 조교가 응답하지 않거나 오류가 발생하는 경우:

  1. 인터넷 연결을 확인하세요.
  2. 페이지를 새로고침해보세요.
  3. 브라우저 캐시를 지우고 다시 시도해보세요.
  4. 문제가 지속되면 지원팀에 문의하세요.
